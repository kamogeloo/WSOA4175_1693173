My idea for an internet art piece is to make css art, to draw a character using css and html, and make it exist in a 3D space, or give the illusion that it exists in a 3D space.

I did actually attempt to do this already, I created an illustration of an actress named Ryan Destiny using css and I attempted to make the illustration look like it exists in a 3D space. I made so tat when a user hovers over it it translates forward and changes color, and when you move your mouse while hovering the illustration will move giving that 3D effect.

Internet art is said to be art that not only uses the internet as "its tool for production and distribution but also as its source material or medium, and exploits or reflects the internet's inherently connective characteristsics."(Cook & Ghidini, 2015). css art matches that definition perfectly, because it does not exist without the internet.
